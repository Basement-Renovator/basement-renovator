Room Format (Repentance):

Header:
	u32 	# of room entries

Room Entry:
	u32		Type
	u32		Variant (ID)
	u32		Subtype
	u8		Difficulty, observed values: 1, 5, 10, 15, 20
	u16		Length of room name string
	str		Room name
	float	Weight (Room)

	u8		Room Width: 13 or 26
	u8		Room Height: 7 or 14
	u8		Room Shape: 1-12
		1 = 1x1,
		2 = 1x0.5,
		3 = 0.5x1,
		4 = 2x1,
		5 = 2x0.5,
		6 = 1x2,
		7 = 0.5x2,
		8 = 2x2,
		9 = Mirrored L Room,
		10 = L Room,
		11 = Mirrored r Room,
		12 = r room 

	u8		Number of Door Entries
	u16		Number of Entity Entries


Door Entry:
	s16		x position
	s16		y position
	u8		boolean: door exists

	Note: only known values for doors so far are (6,-1), (13,3), (6,7), and (-1,3). Unknown if other values will work as expected.

Entity Entries:
	s16		x position
	s16		y position
	u8		Number of entities at position

	for each entity:
		u16		Entity type
		u16		Entity variant
		u16		Subtype 
		float	Weight (Entity), only relevant when stacked
